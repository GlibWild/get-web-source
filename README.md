### 资源助手
#### 申明
##### 本功能为chrome插件，安装插件后可获取网页资源，便于下载、复制。
### <div style="color:#ff0000">注意本插件仅用于学习使用，严禁用于商业用途和非法使用</div>
#### 实现功能

 显示当前页面所有资源

 [✔] 完成图片、图片颜色、背景颜色、字体颜色、background样式的展示。并允许复制
 
 [✔] 提供下载选项
 
 [&nbsp;&nbsp;&nbsp;]  提供文字复制功能
 

#### 安装
1. 打开edge或chrome管理扩展
2. 打开开发人员模式
3. 选择加载解压缩的扩展，选择本插件文件夹路径
4. 即可使用
